# Explorer Shop: E-commerce Storefront | Vanilla JS & Modular Sass | [Live Demo](https://kolonatalie.github.io/sass-vanilla-js-ecommerce-site/)

## Project Overview

This is a responsive, multi-page e-commerce storefront built entirely with **Vanilla JavaScript (ES6+)** and **Modular SCSS**. This project serves as a comprehensive demonstration of expertise in modern frontend markup development, focusing on **Clean Code**, **Performance Optimization**, and scalable CSS architecture.

The design was implemented from a **Figma** template, showcasing strong **Quality Assurance** skills in translating design into functional code.

### Core Features

* **Multi-Page Structure:** Main, Catalog, Product Detail, Shopping Cart, and Info pages.
* **Dynamic Product Rendering:** Products are fetched and rendered dynamically via JavaScript.
* **Persistent Cart Logic:** Shopping Cart data is stored and managed using **Local Storage**.
* **Advanced Filtering/Search:** Implementation of client-side filtering and search functionality.

## ‚öôÔ∏è Project Setup and Scripts

### Prerequisites

* **Node.js** (v18+)
* **npm** installed on your system.

### Installation

1.  **Clone the Repository:**

    `git clone` [`https://github.com/kolonatalie/sass-vanilla-js-ecommerce-site.git`](https://github.com/kolonatalie/sass-vanilla-js-ecommerce-site.git)
    ```bash
    cd sass-vanilla-js-ecommerce-site
    ```
2.  **Install Dependencies:** Installs all required Node.js packages (Sass, ESLint, Stylelint, Live Server, etc.).
    ```bash
    npm install
    ```

### Available Scripts

| Script | Command | Description |
| :--- | :--- | :--- |
| **`npm start`** | `live-server` | Starts the local development server. |
| **`npm run build`** | `sass scss:css --no-source-map --style=expanded` | Compiles the final CSS into the `css` directory, ready for deployment. |
| **`npm run watch-css`** | `sass --watch scss:css --no-source-map --style=expanded` | Watches SCSS files and automatically recompiles CSS upon changes (for development). |
| **`npm run lint`** | `npm run lint:js && npm run lint:css` | Runs all quality checks (JS/CSS) to ensure **Clean Code** standards are met. |
| **`npm run lint:fix`** | `npm run lint:js -- --fix && npm run lint:css -- --fix` | Automatically fixes style and formatting errors where possible. |
